# KIT 소재디자인공학과 · CFMS 연구실 클라우드

금오공대 소재디자인공학과(CFMS Lab) 연구실 시스템을 클라우드에서 운영합니다.
화면(HTML)은 GitHub Pages, 데이터는 Supabase에 저장됩니다.

---

## 1. 홈페이지 접속

- 주소: **https://cfms-lab.github.io/KIT_sodi/**
- 포털에서 세 시스템으로 들어갑니다.
  - **논문 관리 (UPJUK)** — 종류·저자·연도 검색, PDF 열람
  - **수업 시간표 (TimeTable)** — 학기별 시간표 드래그 편성
  - **취업 현황 GIS (JobGIS)** — 졸업생 취업처 지도
- **로그인**: 화면 우측 상단에서 한 번 로그인하면 세 앱에서 공유됩니다.
  - 조회·검색은 로그인 없이 가능합니다.
  - **추가·수정·삭제(편집)는 로그인해야** 됩니다.
- 각 앱 좌측 상단의 **"← 홈"** 으로 포털로 돌아옵니다.

### 구성 파일
| 파일 | 내용 |
|---|---|
| `index.html` | 포털(홈) |
| `upjuk.html` | 논문 관리 |
| `timetable.html` | 수업 시간표 |
| `jobgis.html` | 취업 현황 GIS |
| `backup.html` | 데이터 백업 도구 |

## 2. 수업 시간표 — 과목 다루기

먼저 우측 상단에서 **로그인**하고, 상단에서 **학기**를 선택하세요. 그다음 과목 블록을 이렇게 다룹니다.

- **이동**: 과목 블록을 **드래그**해서 원하는 칸(요일·학년·교시)에 놓습니다. → 자동 저장
- **복사**: **`Alt` 키를 누른 채 드래그**하면 원본은 그대로 두고 사본이 새로 생깁니다. (드래그 중 "＋복사" 표시)
- **삭제**: 과목을 **클릭**해 선택(파란 테두리)한 뒤 **`Delete`(또는 `Backspace`) 키**를 누릅니다.
  - 또는 과목을 **더블클릭** → 뜨는 창의 **삭제** 버튼.
- **수정**: 과목을 **더블클릭**하면 편집 창이 열립니다.
- **추가**: 상단 **`＋ 과목 추가`** 버튼.
- **새 학기 만들기**: 상단 **`학기 복제`** 버튼 → 이전 학기를 통째로 복사해 새 학기를 시작합니다.

> 모든 변경은 그 자리에서 클라우드에 저장되어, 다른 PC에서 새로고침하면 바로 반영됩니다.

## 3. 데이터 · 백업

- 모든 데이터는 **Supabase** 한 곳에 저장됩니다(이 저장소에는 화면 파일만 있습니다).
- 백업은 **`backup.html`** 에서 로그인 후 "복원용 SQL/JSON 받기"로 내려받아 `backup/` 폴더에 커밋합니다.
- 이 저장소는 **공개**이므로, 취업 데이터 백업 시 **"이름 마스킹"** 을 켠 채로 받으세요(학생 실명 노출 방지). 실명 포함 전체 백업은 개인 PC/비공개 저장소에만 보관합니다.
